# **清算陈鸽牧师——一棵从根里坏起的坏树**

本来已经万事交托，安心养老等待回天家享福，谁承想那个十年来没有任何长进的“小草”突然来搅和，打乱平静的心情；并不想多理会她，只客套几句打发算了；对她提到的事情也没有兴趣，是早已解决过的问题：五年前我已经劝过陈鸽牧师收山退隐，静心做神学研究，留下一点有价值的东西传世；因为那时候他讲的信息已经是完全错误，纯属不懂装懂强充内行，不但是误导绊倒人，也是为自己积累罪恶。他既然不听劝，反而向国内教会大肆推荐“武汉骆传道”，也只能任他不听；审判的主自有公断。

前些日子预感到主耶稣可能给了召我回天家的信号，就紧急盘算清点了一下还有什么要赶快做完需要了结的事情，把所有事情做个总结，免得有牵挂。总结下来认为现在是回天家的最佳时机，一身轻松一心无挂，于是静下心来，写了一篇《致谢我爱阿朱阿紫弟兄》，作为对一位弟兄观察试验二十年的结论，希望这个肯定和感谢能带给他一些鼓励，因为他游离于教会保护支持之外长期孤军无援，仍然忠心地坚持着神交托的特别工作，非常不易。

对于陈鸽牧师（Larry Pan），同样是观察了二十年，却不愿再提及，实在不愿再提起这些伤心事。“忧伤的灵使骨枯干；喜乐的心乃是良药”，为了写作完任务的缘故，多年不得不每天面对各种负面灰暗冰冷伤感也没地方说理的事儿，天天咀嚼“琴挂地柳树上”的忧伤，使身体和心灵受到很大亏损。自从被神命令退休停工，就不愿再关注那些垂垂暮年刚硬不弯的傲骨们，只想从不断进步生命日新的年轻小孩身上感受喜乐，感恩神的奇妙眷顾赐福。既然“小草”又挑起事端，不妨顺便也总结一下对陈鸽牧师二十年的观感，作为《致谢我爱阿朱阿紫弟兄》 的对偶篇，迎合一下陈鸽牧师擅长讲的“平衡”，虽不能使小草服气，我自己总算没有挂碍了。

陈鸽牧师坏的地方是在根里，结出歪果子之前茂盛的青枝绿叶很能唬住人，所以能骗得过很多家庭教会的领袖肢体。那时候，又年青，又能讲神学，又热心，又勇敢，声音又大的陈鸽牧师几乎比每一位家庭教会的年青弟兄都优秀，被大家寄予厚望是合情合理的，对那时的陈鸽牧师不感冒才是心眼偏狭别有用心（除非有圣灵的特别警告，当然这种警醒一直都有）。 

因为看上去真的很好，无论是老人们，还是年青人都在陈鸽牧师身上寄予了很高的期望，希望他作反对灵恩派邪灵的旗手，希望他作抵挡三自三势的火炬手，希望他作批判假先知王怡牧师的先锋，等等，但是所有这些人都会或将会以收获失望而告终。陈鸽牧师连自己都反不了，还能指望他去反对别人；他擅长的平衡术总能在关键时候放自己和别人一马。例如，虽然陈鸽很早已对灵恩前辈王永信、当红英雄生命季刊王峙军、纯金偶像唐崇荣博士牧师的软弱虚伪路线颇有微词，似乎想表达被组织出卖了的不满，但这种醒悟不足以摆脱以前的错误根基；除非在新的正确根基上重新建造，断难与旧日遗毒撇清关系。不对自己狠一点，只能仍作人家的刀下鱼肉或枪杆子被玩弄。

被组织出卖抛弃后，陈鸽牧师便表态自己不是改革宗，不是归正宗，甚至什么宗都不是，而是孤胆英雄，个体户，谁也不巴结。这情绪和来勇敢破冰的陈佐人牧师有一拚，陈佐人牧师代表唐氏归正宗到福建神学院演讲授课，要勇破唐崇荣牧师一生不能与三自合作的冰河，被骂和被组织警告处分后，忿忿地表态自己无宗无派，没人生没人养，没受谁差派，是独闯江湖的第一高手，想干啥干啥爱咋的咋的。其实，谁不知道他多年以唐氏宗的二把手为荣为业；同样，陈鸽牧师凭改革宗金招牌打出自己的一片江湖后，吃饭砸锅，又不承认自己是五要点改革宗了。

唐崇荣牧师认为神对同性恋罪恶的最大审判是让他们不能生育而绝后，但他是养了儿子的，不怕绝后，所以敢以身试法。大概陈鸽牧师是有这样就学这样，也不批判强奸惯犯传异端的远志明，因为他没有女儿儿子可牵挂担心。

任何一位汉语教会的牧师传道人，如果不起来揭露传异端邪说二三十年的强奸惯犯远志明，一定也不爱自己的女儿儿子，或者根本没有女儿儿子。他们以为雪片不会落到自己身上，离远志明又远。事实上远志明不仅在美国加拿大华人教会里强奸，也跑到遥远的欧洲，甚至在澳洲、香港汉语教会里出没。如果不打碎远志明所挂的民运和家庭教会的圣洁牌坊，就算远志明先生死了，也会变成恶鬼污鬼觊觎你的女儿儿子，在你儿女身上有权势。这是明摆着的道理和现实。但陈鸽牧师有什么好害怕，这些和他有什么关系，逮谁骂谁的陈鸽牧师不骂远志明，这样就能解释通了。

我多年在北方种田，知道稗子与麦子的区别，青苗时很难分辨；但稗子早熟，麦穗还青绿时，稗子就会提前一个月发白成熟，以便随风把种子撒进田里，免得被一起收割离田绝了种。这样带来的风险就是容易被识别出来，所以我们通常会在麦子成熟前巡视田里，把已经发白的稗子薅出来，堆在田边用火焚烧成灰。虽然国内信徒尚青绿懵懂，但是对陈鸽牧师来说，已经是该薅出来的时候了。 用二十年看一个人，就算没有绝对把握，也相当确实了。

说实话，就算陈鸽牧师能多年骗得了我们，也没什么大不了；我们信靠主，必能被主保守不受亏损，或者补还蝗虫蚂蚱吃掉的。倒是陈鸽牧师自己如果装蒜到底，就亏大了。

近墨者黑，近圣灵者洁。大家谈到陈鸽牧师与内地家庭教会的关系时，常常夸陈鸽牧师的坚定立场和对内地家庭教会的贡献。我认为，作为生长在国外的人，能多年亲身参与体验上世纪末大陆家庭教会的属灵气氛，是神对陈鸽牧师个别的、特别的怜悯和赐福；那时候教会的火热是能把石头烧成宋窑的属灵氛围，虽然假的假，但真的也很真，不像现在无论城里乡下都跟国外学得呵呵、呵呵、呵呵了。当时大陆家庭教会的属灵氛围对年青时代的陈鸽牧师多有熏陶 ，使其多有受益，几乎可以看见他像摩西那样面皮发光。不过这种影响都发生在表面，并没有让他改变到根里、心里。 我猜原因大概与陈鸽牧师是宣教士和无论到哪个教会都是最能讲神学的身份有关，使他觉得自己是“帮助者”，而不是来学习的，以致于他只顾随风招展炫英姿秀身段，枉费了那几年的宝贵时期，终究还是棵歪脖树；被驱逐回外国后，在宽松不信、信不信都一样的环境中，拗劲就越来越释放，越长越歪 ，逐渐不成样子了。

不妨再举个例子吧，要打就往死里打，省得留后患。陈鸽牧师认为扫罗求问交鬼妇人时，是神破例差遣了撒母耳回来，为要挽回扫罗。也就是说，陈鸽牧师和所有假师傅一样，在判断鬼撒母耳真假前，先肯定了鬼撒母耳的鬼话都是真的，完全应验了；这样先相信鬼话，再去分辨真鬼假魂邪灵，无论得到什么结论，都在魔鬼的圈套中，跳不出魔掌。陈鸽牧师不但信鬼话，把鬼撒母耳当真，还认为这是神破例让真撒母耳回来最后一次努力挽回扫罗，说什么“神断不喜悦恶人灭亡”等等，扯了一大片，这当然是强解圣经自取沉沦的作法，无论讲得多么精彩，感人，有力量，都是错误的。经文这里没有要挽回扫罗的意思，而是要显明神对扫罗的审判弃绝。鬼撒母耳的鬼话就是谎话，完全没有应验，仅仅是高明似是仿真，终究是鬼话。这鬼话的手段与伊甸园中欺骗夏娃的蛇的谎言是同样套路伎俩：亚当夏娃的眼明智慧是神赐给的，不是所吃的野果子功效，否则，夏娃就看不见蛇，也没有心智与蛇交流。蛇的话是谎言，而不是事实。

扫罗最大的罪是交鬼。鬼却不说这第一大罪，避重就轻，只说明日你和你众子必与我在一处。时间也不对（扫罗不是次日被杀），也不是所有儿子都死了（伊施巴力，即伊施波设没在战场上被杀。亦施韦也未在战场上。）。  
非对神的话语有根基的不易分辨。魔鬼最擅长用一半的真理来迷惑信徒。抛开神学争辩如何识别呢？从果子就可以识别。鬼撒母耳只叫扫罗仆倒、惧怕、无力，求问鬼不过枉费一场心机，徒劳无益，还叫他极其惊恐。神的话却不是这样，乃是叫人心中平安，有所得，有所靠，有所抓住，能在百般的艰难中大有喜乐。神的话叫你相信、有盼望，鬼的话叫你害怕，失去信心，失去得救把握。
